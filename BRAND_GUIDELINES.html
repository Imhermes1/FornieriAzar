Fornieri & Azar Brand Guidelines

Typography

Fonts

  • Headings: Outfit (weights: 300, 400, 500, 600, 700, 800)
  • Body: Manrope (weights: 300, 400, 500, 600, 700, 800)
  • Fallback: -apple-system, BlinkMacSystemFont, Segoe UI, sans-serif

Type Styles

  • Headings (H1-H6): Uppercase, weight 500, letter-spacing 0.02em–0.03em
  • Body: Normal case, weight 400, letter-spacing 0.01em
  • Navigation: Weight 500, letter-spacing 0.04em
  • Buttons: Uppercase, weight 600, letter-spacing 0.12em
  • Eyebrow/Labels: Uppercase, weight 500, letter-spacing 0.26em

Color Palette

Primary Colors

  • Off Black — #1A1A1A — Primary text, buttons, logo
  • Off White — #FAFAFA — Backgrounds, cards
  • White — #FFFFFF — Pure white backgrounds
  • Black — #000000 — Absolute black accents

Neutral Grays

  • Gray 100 — #F5F5F5
  • Gray 200 — #E8E8E8
  • Gray 300 — #D4D4D4
  • Gray 400 — #A3A3A3
  • Gray 500 — #737373
  • Gray 600 — #525252
  • Gray 700 — #404040
  • Gray 800 — #262626
  • Gray 900 — #171717

Legacy Accent Colors

  • Navy — #050608 — Dark backgrounds
  • Gold — #C8B48B — Accent/CTA
  • Pearl — #FAF9F6 — Page background
  • Stone — #F6F4EF — Warm off-white
  • Slate — #7C7F86 — Subtle text

Design Tokens

Spacing & Layout

  • Max Width: 1320px

  • Topbar Height: clamp(48px, 6vh, 58px)

Border Radius

  • Large: 50px / 100px (pill buttons)
  • Medium: 25px
  • Small: 12px

Shadows

  • Soft: 0 20px 50px rgba(0, 0, 0, 0.08)
  • Card: 0 10px 30px rgba(0, 0, 0, 0.06)
  • Button Hover: 0 8px 20px rgba(0, 0, 0, 0.15)

Transitions

  • Default: 0.32s cubic-bezier(0.4, 0, 0.2, 1)
  • Fast: 0.2s ease

Button Styles

Primary Button

  • Background: Off Black (#1A1A1A)
  • Text: White
  • Border Radius: 100px (pill)

Ghost Button (Light)
